Viewport

The viewport is the user's visible area of a web page. It varies with the device, meaning it will be smaller on a mobile phone than on a computer screen. It dictates how the browser will scale and size the page content.

Flexbox

The Flexible Box Layout Module, or Flexbox, is a one-dimensional layout model. It makes it easier to design a flexible responsive layout structure without using float or positioning, distributing space and aligning content along a single row or column.

Grid

The CSS Grid Layout Module is a two-dimensional layout system. It allows developers to create complex and responsive web designs by arranging content into both rows and columns simultaneously.
